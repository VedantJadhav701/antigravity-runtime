# Runtime Platform

The local-first AI execution runtime for developers.

## 🚀 Vision
A downloadable desktop application + local runtime that builds applications from prompts.

## 🏗️ Architecture
- **Apps**: Desktop (Tauri) & Landing Page.
- **Runtime**: FastAPI-based core with orchestrator, planner, and executor.
- **Environment**: Sandbox and runtime detector.

## 🛠️ Development
```bash
npm install
npm run dev
```
